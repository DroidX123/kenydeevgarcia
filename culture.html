Journal Entry #1: Keyboard Blitz


On August 20, 2024, one of the CCS Founders Week events named “Keyboard Blitz” happened at CC10. This event is organized by the College of Computer Studies and it’s hosted by Confederates Student Council. For this event, I am part of Attendance Committee.

This event is all about who can type the fastest with the least number of errors on monkeytype.com. There were a considerable amount of people participating in this event, to the point where most of the computers were occupied. During the event, it amazes me how people can type so fast with little to no errors.

After the event, I have realized how fast a person can type. As an aspiring game or software developer, I may want to learn how to type fast so that I can be efficient in typing my programming and coding. I hope that these participants can improve their skills so that they can become efficient in programming and coding too.

Because of this activity, I want to learn how to type faster now.
